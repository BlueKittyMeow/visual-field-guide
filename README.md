# Visual Field Guide

A living visual research repository for developing Lara's personal aesthetic system.

This is not a conventional commercial brand exercise. It is a field guide to a coherent visual world: scholarly archival, specimen-oriented, domestic handmade, optical and scientific, tactile, and substantially fin-de-siècle.

The uncanny is treated as a separate overlay rather than the default identity. The base world is ordered, intimate, observant, materially rich, and quietly obsessive. In uncanny mode, that same world dims, degrades, ghosts, contaminates, and begins to lose coherence.

## Working spaces

- `field-guide/` — durable language, decisions, vocabulary, and exclusions
- `boards/collection-table/` — broad image gathering before premature categorization
- `boards/uncanny-overlay/` — the separate degradation and destabilization layer
- `references/` — sourced or generated visual references
- `catalog/` — metadata and decisions about collected images
- `outputs/` — contact sheets, moodboards, and eventual design handoff material
- `scripts/` — utilities for contact sheets, cataloging, and other production work

## Working principle

Collect broadly, describe precisely, and categorize only when the material itself begins to reveal stable groups.
