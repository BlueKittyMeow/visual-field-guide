# Collection Table

The broad receiving surface for visual references before stable categories emerge.

Place candidate images here without forcing them immediately into typography, material, object, color, or composition boards. Each item should eventually have enough metadata to recover where it came from and enough description to explain why it entered the collection.

Useful questions:

- What exactly is compelling here: object, material, light, arrangement, scale, label, or atmosphere?
- Does it feel lived and used, or staged as an aesthetic commodity?
- Is it autobiographically resonant, historically useful, formally useful, or merely adjacent?
- Would it still belong without sepia, darkness, or age effects?

Subdirectories may be created once repeated visual families become unmistakable.
