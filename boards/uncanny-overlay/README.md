# Uncanny Overlay Board

References in this board should demonstrate transformations that can be applied to the established core world: dimming, ghosting, misregistration, contamination, degradation, duplication, and taxonomic instability.

The board should not become a general collection of creepy images. A strong candidate retains a visible underlying system of order and lets the viewer feel that system beginning to fail.

Keep notes on both the base image and the transformation. This will eventually support paired examples showing ordinary and activated states of the same visual world.
